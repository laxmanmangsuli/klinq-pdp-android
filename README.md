# Klinq Product Detail Task

Android e-commerce product detail screen implementation.

## Features
- Retrofit API integration
- MVVM architecture
- ViewPager2 product image slider
- Color variant selection
- Dynamic image update on color change
- Quantity selector
- Product description expand/collapse
- Share functionality
- Loading & error handling
- ViewBinding + DiffUtil + Coil

## Tech Stack
Kotlin, MVVM, Retrofit, Coroutines, StateFlow, RecyclerView, ViewPager2, Coil

## API
https://klinq.com/rest/V1/productdetails/6701/253620?lang=en&store=KWD

## Run
1. Open project in Android Studio
2. Sync Gradle
3. Run app
